这是谢伟尔钢铁缺陷检测的数据集

1.1 数据集基础信息

- 数据集名称：severstal-steel-defect-detection
- 数据集来源：Kaggle竞赛官网（[官网链接](https://www.kaggle.com/c/severstal-steel-defect-detection/data)）
- 数据集应用场景：工业场景下的钢铁表面缺陷分割任务，涵盖4类常见钢铁表面缺陷
- 原始数据集格式：
    - 图片：256×1600像素的JPG格式灰度图（钢铁表面扫描图）
    - 标注：CSV格式的像素级编码（Run-Length Encoding，RLE）
- 数据量：
    - 原始图片总数：12568张

1.3 数据集预处理流程

1.3.1 处理目标

将原始的REL编码标注转换为YOLOv8 Segmentation支持的格式。

1.3.2 格式转换具体步骤（[参考文献](https://blog.csdn.net/G_Shengn/article/details/137969275?fromshare=blogdetail&sharetype=blogdetail&sharerId=137969275&sharerefer=PC&sharesource=2301_76327397&sharefrom=from_link)）

1、RLE编码转掩码

- 处理逻辑：首先解析 RLE 编码字符串，拆分出 “起始像素索引” 和 “连续像素长度” 两组核心参数；以原始图片尺寸 256×1600 为基准，按行优先（row-major）的像素排列规则，将编码对应的像素位置标记为缺陷区域（值为 1），其余背景区域标记为 0，最终生成与原图尺寸完全一致的单通道二值掩码图
- 输出格式：每个独立的缺陷标注对应 1 张 8 位深度的 PNG 格式掩码图

2、标注格式转换

- YOLO Seg 标注文件生成：针对每张图片，整合其所有缺陷的掩码信息，生成 1 个与图片同名的.txt标注文件；文件内每行对应 1 个缺陷实例，格式严格遵循 YOLOv8 Seg 官方规范

1.3.3 数据集划分

- 划分策略：8:2比列随机划分训练集/测试集
- 划分结果：
    - 训练集：10054张图片，对应标注文件10054个
    - 测试集：2514张图片，对应标注文件2514个
